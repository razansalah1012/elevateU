# elevateU 

elevateU is a modern online job portal platform developed as a mini project for **Web Programming**.  
The system connects **job seekers** and **employers** through a user-friendly web platform where users can build profiles, search for jobs, and apply online.

---

## Project Overview

elevateU was designed to simplify the recruitment process by providing:

- A platform for users to discover job opportunities
- A dashboard for employers to post and manage jobs
- Online applications with database storage
- Professional user profile management
- Responsive and modern web interface

## ✨ Features

## User Features

- Register account
- Login / Logout
- View personal dashboard
- Edit profile
- Upload profile image
- Add skills / bio / address
- Browse available jobs
- Apply for jobs
- View submitted applications

---

## Employer Features

- Employer login
- Post new jobs
- Edit job listings
- Delete job listings
- View applicants
- Accept / Reject applications

## Admin Features

- Monitor job listings
- System overview dashboard
